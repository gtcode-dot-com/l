## About Tiktoken Ex

Pure-Elixir TikToken-style byte-level BPE tokenizer, with helpers for Kimi K2 encodings.

## Installation

Add `tiktoken_ex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {'tiktoken_ex', "~> 0.1.0"}\n  ]
end
```

Then run:

```bash
mix deps.get
```
