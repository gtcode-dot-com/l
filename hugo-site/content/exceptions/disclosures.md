title: "Context and Provenance"
description: "A short note on context, provenance, and traceable technical work."
---

Technical work is easier to trust when provenance is visible. Inputs, assumptions, authorship, transformations, and review state should travel with the output.

That principle shows up across GTCode's AI work: source-aware retrieval, reproducible experiments, agent instructions, evaluation manifests, and documentation that explains what a system is allowed to do.
