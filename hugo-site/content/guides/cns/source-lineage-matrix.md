## What CNS 8.0 deletes

CNS 8.0 deletes the architecture shape in which possible-world ranking or access-state analysis becomes the main mechanism.

## What CNS 8.0 demotes

CNS 8.0 demotes any grounding subsystem name that competes with Chiral Narrative Synthesis. Grounding is a substrate. CNS is the synthesis system.

## What CNS 8.0 adds

CNS 8.0 adds an explicit orthesis acceptance protocol combining:

- proof trace status;
- language–logic round-trip residual;
- residual contradiction energy;
- topology diagnostics;
- access-state disclosures;
- calibrated possible-world uncertainty.
